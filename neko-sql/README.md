# @devbro/neko-sql

SQL engine for quick abstracted relational database connections.

## supported features

- data manipulation(select, insert, update, delete)
- schema manipulation(create table, alter table, ...)
- transactions
- joins

## Suported Databases

currently supported databases:

- postgresql

future planned:

- mysql
- sqlite
- mssql
